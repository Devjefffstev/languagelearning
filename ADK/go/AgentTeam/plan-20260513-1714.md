# Agent Team – Go ADK Implementation Plan

## Problem
Build a multi-agent system in Go using Google ADK that orchestrates specialized sub-agents
via different LLM backends (Gemini and Ollama via litellm).

## Approach
- Use `google.golang.org/adk` for agent framework (llmagent, functiontool, geminitool)
- Implement a custom `model.LLM` adapter wrapping `github.com/voocel/litellm` (OpenAI-compat) for Ollama
- Wire agents together with SubAgents in the orchestrator

## Agents
| Agent            | Model              | Backend             | Tools         |
|------------------|--------------------|---------------------|---------------|
| orchestrator     | gemini-2.5-flash   | Google AI           | GoogleSearch  |
| weather_agent_gog| gemma4:e2b         | Ollama local :11434 | get_weather   |
| greeting_agent   | llama3.2:1b        | Ollama local :11434 | say_hello     |
| farewell_agent   | gemma4:31b-cloud   | Ollama cloud        | say_goodbye   |
